StandardProxy 

abstract class StandardProxy[S, RP, A] extends Object with Proxy[S, RP, A]

Create an object to pass to reactCreateClass. Since the actual definition of SLF is client defined you still need to define a mkSelf function (and displayName). This is an interop class that when created, creates a js class, but we only care about it as a js.Object. Hence methods that will get picked up by reactCreateClass are vals and ThisFunctions. Other methods on the trait are fine and turn into class methods. This approch allows easier extensibility e.g. override some of the abstract types in subclasses.
