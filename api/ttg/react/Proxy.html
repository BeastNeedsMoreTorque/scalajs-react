Proxy 

trait Proxy[S, RP, A] extends Object

Data structure used to create js side react class. It's only used once in scalareact.createClass. Most of the methods below recover the scala side "element" (the component spec) and then forwards the call on this "class" to the scala side "element." Thees methods are only called by the proxy/shim object to hook up the internal API.
